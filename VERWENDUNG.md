# Verwendungsanleitung - Puch Maxi Geschwindigkeitsrechner

## So funktioniert der automatische Setup-Vorschlag

### 1. Wunschgeschwindigkeit eingeben

Gib einfach deine gewünschte Zielgeschwindigkeit in das Feld "Wunschgeschwindigkeit" ein.

**Beispiel:** Du gibst `50` ein.

### 2. Automatische Auswahl

Das System erkennt automatisch den passenden Geschwindigkeitsbereich und:

✅ **Wählt das beste Setup aus den vordefinierten Konfigurationen**
✅ **Stellt alle Dropdowns automatisch ein** (Zylinder, Vergaser, Übersetzung, Auspuff)
✅ **Zeigt dir die berechnete Endgeschwindigkeit**
✅ **Zeigt alle verfügbaren Setup-Varianten für diesen Bereich**

### 3. Alternative Setups

Wenn es mehrere Möglichkeiten gibt (z.B. bei 50 km/h), siehst du:
- Das **aktiv angewendete Setup** (mit grünem ✓ Badge)
- **Alternative Setups** mit Button "Dieses Setup anwenden"

Klicke einfach auf den Button, um ein alternatives Setup auszuprobieren.

## Geschwindigkeitsbereiche

Das System unterstützt folgende Bereiche:

| Eingabe | Bereich | Verfügbare Setups |
|---------|---------|-------------------|
| 0-37 km/h | 35 km/h | 1 Setup (Original) |
| 38-42 km/h | 40 km/h | 1 Setup |
| 43-47 km/h | 45 km/h | 1 Setup |
| 48-52 km/h | 50 km/h | **2 Setups** (50ccm oder 65ccm) |
| 53-62 km/h | 55-60 km/h | **3 Setups** (50/65/70ccm) |
| 63-72 km/h | 65-70 km/h | 1 Setup |
| 73-78 km/h | 75 km/h | 1 Setup |
| 79+ km/h | 80+ km/h | 1 Setup |

## Beispiel-Workflow

**Ziel: 55 km/h erreichen**

1. Gib `55` in das Wunschgeschwindigkeit-Feld ein
2. Das System wählt automatisch: **50ccm / 15mm / 15/45 / 28mm**
3. Du siehst 3 alternative Setups:
   - Variante 1 (50ccm) - **Aktiv** ✓
   - Variante 2 (65ccm) - Button "Anwenden"
   - Variante 3 (70ccm) - Button "Anwenden"
4. Klicke auf ein alternatives Setup um es auszuprobieren
5. Die Dropdowns werden automatisch angepasst
6. Die Geschwindigkeit wird neu berechnet

## Sicherheitswarnungen

Das System warnt dich automatisch bei gefährlichen Kombinationen:

⚠️ **28er Reso Auspuff** mit kleinen Zylindern (≤65ccm)
⚠️ **18er Auspuff** mit großen Zylindern (≥65ccm)
⚠️ **Übergroße Vergaser** (>19mm) mit kleinen Zylindern (≤70ccm)

## Manuelle Anpassungen

Du kannst jederzeit die Dropdowns manuell ändern:
- Die Geschwindigkeit wird sofort neu berechnet
- Warnungen werden aktualisiert
- Die Vorschläge bleiben sichtbar

## Tipps

- **Start einfach:** Beginne mit niedrigeren Geschwindigkeiten (40-50 km/h)
- **Prüfe Warnungen:** Beachte immer die Sicherheitswarnungen
- **Teste Varianten:** Probiere verschiedene Setups aus dem gleichen Bereich
- **Vergleiche:** Nutze die angezeigten Setups als Orientierung für deinen Kauf

## Vorteile der vordefinierten Setups

✅ Bewährte Kombinationen aus der Praxis
✅ Abgestimmt auf realistische Zielgeschwindigkeiten
✅ Mehrere Varianten für unterschiedliche Budgets
✅ Sicherheit durch geprüfte Konfigurationen
✅ Sofortige Anwendung ohne komplizierte Berechnungen
